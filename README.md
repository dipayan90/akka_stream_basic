# Basic Akka Streams Introduction Project

A simple basic Akka stream poc

#Prerequisites

-- Java JDK 1.8
-- Scala 2.11.6
-- SBT

#Building and Running Project

From Root folder : 

sbt compile
sbt run 

#Project Inclusions

1. Akka Basic Flow
2. Akka Multiple Sources Flow
3. Akka split and merge Flow
4. Akka streams variable time Flow
